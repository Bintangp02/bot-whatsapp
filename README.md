## Clone this project

```bash
> git clone https://github.com/Bintangp02/whatsapp-bot
```

## Install the dependencies:
Before running the below command, make sure you're in the project directory that
you've just cloned!!

```bash
> cd whatsapp-bot
> bash install.sh
```

### Usage
```bash
> npm start
```
